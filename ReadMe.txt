Primeiros Passos

Criar sua pasta manualmente em sistema operacional.

Apos a Instalacao do GitHub https://git-scm.com/downloads

Abrir a sua pasta e com o Botao  direito clicar em Git Bash Here

Agora o prompt de comando git esta com o endereco de sua pasta.

Para que o Git reconheca e trabalhe com essa pasta, voce dee realizar o primeiro comando.

git init 

Sera criada uma pasto oculta dentro da sua pasta, pois sao os arquivos de controle do github. 

Agora vamos criar um arquivo chamado readme.txt (como eu criei este que voce esta lendo)

muito simples, basta digitar echo reade > readme.txt e enter. 

sempre sera o ECHO nome > nomedoarquivocomextensao
